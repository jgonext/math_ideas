[v2]

Pantalla "Configurar Nuevo Juego":
- Hay que compactar los elementos de configuración para evitar que el usuario tenga que scrollar
- El formato de cada elemento de configuración será lineal:
	- Opción 1: Selección | Entrada de usuario
	- Opción 2: Selección | Entrada de usuario
	- etc
- Opción "Configurar Nuevo Juego": se quitan los nombres de animales y la imagen del avatar se muestra dentro de un circulo

Pantalla del Juego:
- Se agrega un botón "Volver" igual que en la pantalla de historial, pero en esta se pide confirmación para volver 
y en caso afirmativo no se guarda nada esa partida no cuenta.

Pantalla Historial:
- El listado tiene una cabecera unica al inicio y final del listado, y cada elemento del listado debe ser más compacto
- Cada elemento del listado muestraba:
	- Fecha del juego
	- Codigo usuario
	- Tipo de juego
	- Nivel
	- Aciertos -> X / Y  (ejemplo: 2 / 3)
	- Puntos obtenidos en ese juego
	- Total de puntos acumulados en ese momento
	- Tiempo
	- Botón para eliminar elemento

[v3]

Pantalla "Juego Finalizado":
- Botón "Nuevo Juego" se renombra por "Repite Juego", en tal caso se repite con la misma configuración un nuevo juego,
es decir, que no es necesario volver a pedir al usuario la configuración.

Pantalla "Configurar Nuevo Juego":
- Opción "Código" pasa a ser la primera
- Opción "Avatar", que será la segunda, si el usuario a introducido un codigo que ya exista 
se le pre-asignará el último avatar que el usuario eligió. Aún así, luego lo podría cambiar de nuevo.

Pantalla de los Juegos de operaciones:
- En los cuadros solo se debe permitir introducción de números.
- Si el usuario dentro del cuadro, si pulsa:
	- flecha izquierda pasará al cuadro de la izquierda si existe
	- flecha derecha pasará al cuadro de la derecha si existe

Pantalla Historial:
- Aunque se eliminen elementos del listado el total de puntos obtenido en todas las partidas no se perderá
